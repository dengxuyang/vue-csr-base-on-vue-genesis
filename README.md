## 介绍
这是一个基于 Vue SSR Genesis 框架快速开发的模板例子<b>的例子</b> 里面有一个前端 一个后端 font只是一个简单的脚手架

实现了远程组件的使用，elementui的集成，前端只需要安装远程组件后，调用接口，至此我们就不必要用ifream，嵌入一个页面了！

## 启动
```base
# 安装依赖
yarn
# 开发
yarn dev
# 编译
yarn build
# 运行编译后的程序
yarn start

```
